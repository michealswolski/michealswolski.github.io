# michealwolski.github.io
> Personal cybersecurity portfolio — Live at [michealwolski.github.io](https://michealwolski.github.io)

## Features
- Liquid chrome canvas background with iridescent metaballs
- Terminal boot sequence on first load
- Dark/light theme toggle with smooth transitions
- Project detail modals with expanded descriptions
- Interactive timeline for work experience
- GitHub contribution heatmap
- Magnetic cursor effect on buttons
- Parallax scrolling between sections
- Scroll-triggered reveal animations on every element
- 23+ projects across 6 categories: Internship, Coursework, Personal, GitHub, Labs

## Tech Stack
- React 18 + Vite
- GitHub Pages (auto-deployed via GitHub Actions)
- No external CSS frameworks — all custom styling

## Local Development
```bash
npm install
npm run dev
```

## Deploy
Push to `main` — GitHub Actions automatically builds and deploys to Pages.
